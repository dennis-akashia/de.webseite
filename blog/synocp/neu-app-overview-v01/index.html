🚀 Release: SynoCP App Overview v0.1

Heute veröffentliche ich mit SynoCP App Overview die erste öffentliche Komponente des SynoCP-Projekts.

Die Anwendung ist ein leichtgewichtiges PHP-Dashboard für Self-Hosted Umgebungen. Ziel ist es, eine zentrale Übersicht für alle Anwendungen auf einem Server bereitzustellen, ohne dass dafür manuell eine Liste gepflegt werden muss.

Die Idee
Viele Self-Hosted Setups bestehen aus mehreren kleinen Tools, Admin-Oberflächen oder internen Anwendungen. Mit der Zeit entsteht eine Sammlung von URLs, die irgendwo dokumentiert sind – oder auch nicht.

SynoCP App Overview löst dieses Problem über automatische Discovery.

Jede Anwendung enthält lediglich eine kleine Konfigurationsdatei: app.json. Das Dashboard scannt die Unterverzeichnisse, liest diese Dateien zur Laufzeit ein und generiert daraus automatisch eine Übersicht.

Neue Anwendungen erscheinen damit automatisch im Dashboard.

Funktionen

Auto-DiscoveryUnterverzeichnisse werden nach app.json Dateien durchsucht.
KategorienApps können über ein category Feld gruppiert werden.
FavoritenMit "favorite": true markierte Anwendungen erscheinen in einem separaten Bereich.
Zuletzt verwendete AppsDie letzten sechs geöffneten Anwendungen werden im Browser über localStorage gespeichert.
Live-SucheApps können direkt nach Name oder Beschreibung gefiltert werden.
Dark / Light ThemeEin Theme-Toggle ermöglicht den schnellen Wechsel. Die Einstellung wird ebenfalls im Browser gespeichert.
Login-IndikatorApps mit "login_required": true werden mit einem 🔒 Symbol gekennzeichnet.
Custom IconsAnwendungen können eigene Icons bereitstellen. Falls keines vorhanden ist, wird automatisch ein Fallback-SVG verwendet.
Optionales CachingDie generierte App-Liste kann optional in eine JSON-Datei geschrieben werden, um Dateisystemzugriffe zu reduzieren.
Responsives LayoutDas Karten-Grid passt sich kleineren Displays an und zeigt auf mobilen Geräten zwei Spalten.

Einsatzmöglichkeiten
SynoCP App Overview kann auf zwei Arten genutzt werden:
- als eigene Intranet-Startseite für Self-Hosted Anwendungen
- als Unterordner innerhalb einer bestehenden Anwendung
Dadurch lässt sich die Übersicht relativ einfach in bestehende Umgebungen integrieren.

Feedback und Anregungen sind willkommen.
